Computer science: Data

Using binary

Binary is used to represent data and instructions in a program.

But what's so special about binary?
Computer processors contain billions of transistors (like a switch) that only have two states: on and off (like a switch again).
Because there are only 2 states, binary is used to represent these states and thus determine what the computer should do.

The binary system
The two states in binary are represented by 1 (on) and 0 (off). These are called 'binary digits' or 'bits'. All instructions and data including graphics and sound are represented as millions of bits.

Why are binary numbers so long?
Because binary only has two numbers, you need multiple digits to symbolise the same value. (For example, 42, while only needing 2 digits in 'standard' numbering, requires 6 digits in binary numbering [101010]) This results in a generally longer number, when compared to other forms of numbering.

Logical shifts

What are they?
Logical shifts are used when you want to multiply or divide binary numbers by a number which is a power of 2. (Including: 2, 4, 8, 16, 32, 64, etc) Logical shifts aren't just restricted to binary though, it is also used in decimal, except what you do is multiply or divide by a number which is a power of 10. (Including: 10, 100, 1000, etc.)
